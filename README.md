
# Prim Hesaplama Aracı

Bu çalışma, kişisel mühendislik portfolyo web sitemin kaynak kodlarını içermektedir. Site; mekatronik mühendisliği geçmişimi, endüstriyel otomasyon alanındaki saha deneyimimi, PLC programlama ve endüstriyel robot entegrasyonu çalışmalarımı profesyonel bir şekilde sunmak amacıyla hazırlanmıştır.

Portfolyo; eğitim bilgilerimi, teknik yetkinliklerimi, görev aldığım endüstriyel otomasyon projelerini, sertifikalarımı ve iletişim bilgilerimi tek bir dijital profil altında toplar.

## Projenin Amacı

Bu web sitesinin temel amacı, mühendislik deneyimimi yalnızca klasik bir CV formatıyla değil, daha anlaşılır, erişilebilir ve görsel olarak düzenli bir web arayüzü üzerinden sunmaktır.

Site; iş başvurularında, profesyonel görüşmelerde ve teknik yetkinliklerimi paylaşmam gereken durumlarda destekleyici bir dijital portfolyo olarak kullanılmak üzere geliştirilmiştir.
